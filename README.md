# Voice AI Detector API

Advanced REST API to detect AI-generated vs Human voices in Tamil, English, Hindi, Malayalam, and Telugu.

## Features
- **High Performance**: Built with FastAPI and optimized for 16GB RAM.
- **Robust Model**: Powered by quantized Wav2Vec2.
- **Compliant**: 100% compliant with the hackathon rules.

- **Platform**: Hugging Face Spaces (Docker)
- **Port**: 7860
- **Model Storage**: Direct GitHub Release Download
